# DESIGN.md

## 气质与意象
深夜卧室里，两个人背对背坐在床上，手机屏幕的微光照亮各自的侧脸——空气紧张但不绝望。整个界面应该像微信聊天窗口一样亲切熟悉，让用户觉得"这就是我每天用的东西"。

## 配色方案
- 背景色：#EDEDED（微信聊天背景灰）
- 对方气泡：#FFFFFF（白色，左侧）
- 用户气泡：#95EC69（微信绿，右侧）
- 好感度条：从 #FF4D4F（愤怒红）渐变到 #95EC69（开心绿）
- 加分动画：#52C41A 绿色上升
- 减分动画：#FF4D4F 红色下降
- 强调色：#07C160（微信品牌绿）
- 文字色：#1A1A1A 主文字，#999999 辅助文字

## 字体排版
- 字体族：PingFang SC, Hiragino Sans GB, Microsoft YaHei, sans-serif
- 气泡内文字：15px / 1.5
- 选项文字：14px / 1.4
- 进度条标签：12px

## 页面结构
- 顶部固定：好感度进度条 + 轮次指示器 + 声音播放按钮
- 中间滚动：聊天气泡区域（对方左/用户右，带头像）
- 底部固定：6个选项按钮（2列3行布局，移动端友好）

## 动效与交互
- 气泡出现：从下方滑入 + 淡入，150ms
- 好感度变化：数字跳动 + 进度条平滑过渡，300ms ease-out
- 加分：绿色 "+N" 从进度条位置上浮消失
- 减分：红色 "-N" 从进度条位置下沉消失
- 选项点击：按压缩放反馈，100ms
- 通关：全屏撒花粒子效果
- 失败：心碎动画 + 屏幕轻微震动

## 设计禁忌
- 不使用冷色科技风格（蓝紫色渐变等）
- 不使用圆角大于 16px 的卡片式布局
- 不在气泡内使用 emoji 作为装饰
- 进度条不显示具体加减分数字给用户
